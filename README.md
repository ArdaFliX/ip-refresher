# ip-refresher
Refresh your IP and DNS cache with this tool!

![Screenshot of the tool.](https://i.imgur.com/g7dMBeD.png)


### Functions

_**Updates the DNS cache.**_

_**Obtains a new IP address to use on the local network.**_

_**Clears DNS cache.**_
